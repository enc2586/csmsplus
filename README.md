# PDF 다운로더 확장 프로그램

특정 온라인 문서 뷰어에서 PDF 파일을 쉽게 다운로드할 수 있는 Chrome/Edge 확장 프로그램입니다.

## 설치 방법

1. Chrome에서 `chrome://extensions` 접속
2. 우측 상단 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 폴더 선택

## 사용 방법

1. 지원되는 문서 뷰어 페이지 방문
2. 우하단에 나타나는 검은색 원형 다운로드 버튼 클릭
3. PDF 자동 다운로드 완료

## 기능

- 자동 페이지 감지
- 원클릭 다운로드
- 실시간 진행 상황 표시
- 자동 파일명 설정

## 기술 스택

- Manifest V3
- pdf-lib
- Background Service Worker
